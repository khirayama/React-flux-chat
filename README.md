# Fluxチュートリアルのchatをgulpとwebpackで動かすぉ
## 実行手順
下記を実行してgulpとcoffee-scriptをグローバルにインストール。
nodeがない場合は[ここ](https://nodejs.org/)を参考にインストールしておきましょう！

```
$ sudo npm install -g gulp coffee-script
```

次にモジュールをインストール！

```
$ sudo npm install
```

そしたらgulpを起動して、アプリを立ち上げて完了！

```
$ gulp
```
